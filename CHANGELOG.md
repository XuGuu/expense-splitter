# 更新日志

记录这个项目的每次更新。版本号用 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.2.0] - 2026-06-05

### 新增
- 📅 每笔账目支持记录日期（默认填今天）；账目列表和分享文本中显示
- 🏷 GitHub 主题标签（split-bill、expense-tracker 等）方便被搜到
- 🏅 README 加 MIT + 零依赖徽章

## [1.1.0] - 2026-06-04

### 新增
- ✏️ 编辑账目：点账目上的笔图标可修改而不必删了重建
- ✏️ 双击成员名字可重命名（所有账目里的引用自动同步）
- 📋 一键复制结算文本：直接发到群里通知大家
- 🧹 全部清空：一段旅程结清后开始下一趟

### 改进
- 更新 LICENSE 中的版权人信息
- 更新截图，展示新功能

## [1.0.0] - 2026-06-04

### 新增
- 添加成员、记录每笔支出
- 自动结算每个人的「应收回 / 需支出」
- 用贪心算法给出转账次数最少的还款方案
- 数据自动保存（浏览器本地存储）
